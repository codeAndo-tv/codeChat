## 💬 ¿Qué es codeChat?

**codeChat** es el proyecto que construimos juntos en el curso **Angular desde Cero** de mi canal de YouTube [**codeAndo_**](https://youtube.com/@codeAndo_).

Vamos a crear un chat completo con:

- 🔐 Pantalla de **Login** con formularios reactivos
- 💬 Pantalla de **Chat** con barra lateral y mensajes
- 🧭 **Navegación** entre pantallas sin recargar
- 🎨 **Diseño oscuro** profesional explicado línea por línea

No es teoría vacía. Es **aprender haciendo.**

## 🚀 ¿Cómo ejecutar codeChat en tu máquina?

```bash
# 1. Cloná el repositorio
git clone https://github.com/codeAndo-tv/codeChat.git

# 2. Entrá a la carpeta
cd codeChat

# 3. Instalá las dependencias
npm install

# 4. Ejecutá el proyecto
ng serve --port 5300
